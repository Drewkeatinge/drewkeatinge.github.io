Javascript DOM

First of all, What is it?

Javascript is a programming language that we can use to manipulate webpages.

The DOM (Document Object Module) is an interface that allows content on a webpage to be created or changed.

For examples the Header and menus on this page are all created using Javascript

How does it all work?

Javascript uses a sort of naming convention to find and target the building block of a website - we call these names classes and/or ids. Using these 'names' we can ask the relative blocks to behave in certain ways.

How might you interact with it?

Arrays vs Objects

Arrays store data in a singular type fashion i.e (1, 2, 3, 4, 5), you can access this information by referencing the sequance location whereas an object stores things as key-value pairs i.e {PE:Ball, Math:Ruler}. To access the value you need to first address the key.

Javascript in relation to HTML & CSS

Best way I can describe this is with a set of drawers. HTML is what the drawers are made of and also what each draw contains. CSS is how the drawers look and feel. Javascript is the action of opening and or closing the drawers.
